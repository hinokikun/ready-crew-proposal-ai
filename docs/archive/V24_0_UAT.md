# Version 24.0 UAT

## 使用データ

実在企業、実メールアドレス、実電話番号、Password、APIキー、Token、顧客本文全文は使用しないでください。架空会社とテスト用メモだけを使用します。

## 必須確認

| No | 確認項目 | 期待結果 | 結果 |
| --- | --- | --- | --- |
| 1 | adminログイン | 管理者メニューを利用できる |  |
| 2 | managerログイン | レビュー・履歴・運用確認ができ、ユーザー作成はできない |  |
| 3 | memberログイン | 案件作成、AI生成、PPTX/PDF/Beautiful.ai出力ができる |  |
| 4 | viewerログイン | 閲覧のみで生成・出力・編集はできない |  |
| 5 | 新規member作成 | ユーザー一覧へ表示され、Passwordは表示されない |  |
| 6 | Password再設定 | 仮Passwordでログインでき、変更必須フラグが表示される |  |
| 7 | 自分のPassword変更 | 変更後に再ログインが必要になる |  |
| 8 | 無効ユーザー拒否 | ログインできない |  |
| 9 | Pilot対象外拒否 | Pilot Mode中は対象外ユーザーが拒否される |  |
| 10 | 案件作成 | STEP 1からAI生成へ進める |  |
| 11 | AI生成 | 提案内容が表示される |  |
| 12 | Quality Gate | 未確認数が減り、全チェック後に完了できる |  |
| 13 | 要約PPTX | ダウンロードできる |  |
| 14 | 詳細PPTX | ダウンロードできる |  |
| 15 | 見積PDF | ダウンロードできる |  |
| 16 | Beautiful.ai | 成功時にリンクが表示され、失敗時は理由が表示される |  |
| 17 | 作成履歴 | AI生成、PPTX/PDF、Beautiful.ai履歴が確認できる |  |
| 18 | 再出力 | 履歴または現在案件から再出力できる |  |
| 19 | 監査ログ | ログイン、ユーザー管理、生成、出力、エラーが残る |  |
| 20 | Organization跨ぎ拒否 | 他Organizationのデータへアクセスできない |  |
| 21 | Workspace跨ぎ拒否 | 他Workspaceのデータへアクセスできない |  |
| 22 | Logout | 認証情報が削除される |  |
| 23 | Token期限切れ | ログイン画面へ戻る |  |
| 24 | API timeout | 無限loadingにならずエラー案内が出る |  |
| 25 | request_id表示 | エラー時に管理者へ伝える情報を確認できる |  |
| 26 | 360pxモバイル | 横スクロールなしで主要操作ができる |  |

重大項目が失敗した場合は本番利用不可です。重大項目はログイン、案件作成、AI生成、Quality Gate、出力、権限、Workspace分離、Healthです。
