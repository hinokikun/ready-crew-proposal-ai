# スクリーンショット撮影一覧

## 目的

UAT終了後にWordマニュアルへ挿入するスクリーンショットの候補一覧です。

撮影時は架空データを使用し、実顧客名、実メールアドレス、電話番号、APIキー、パスワード、機密案件本文を含めないでください。

## 撮影ルール

- ブラウザ幅はPC 1440pxを基本とします。
- モバイル確認用は360px幅で撮影します。
- Organization、Workspace、Roleが分かる状態で撮影します。
- 管理者画面はadmin権限で撮影します。
- 一般利用者画面はmember権限で撮影します。
