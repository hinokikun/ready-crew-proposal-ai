# Quality Model

The Proposal Quality Evaluator scores ten categories.

| Category | Question |
|---|---|
| Story Consistency | Does the story type match the Presentation Context and slide plan? |
| Persona Alignment | Is the proposal aimed at the right audience and decision maker? |
| Strategy Alignment | Does the selected strategy match the chosen presentation pack? |
| Message Clarity | Is the main message short, clear, and supported by priority messages? |
| Evidence Quality | Are evidence levels sufficient for review? |
| KPI Quality | Is KPI coverage explicit and measurable? |
| Risk Coverage | Are proposal risks identified and represented in the slide plan? |
| Roadmap Completeness | Are roadmap, PoC, timeline, or next-step structures present? |
| Presentation Pack Consistency | Does the Presentation Context preserve the selected pack? |
| Call To Action | Are next actions concrete enough for the sales owner? |

## Evaluation Principle

The evaluator does not rewrite proposal content.

It produces:

- score
- reason
- improvement suggestion
- red flag

The report is intended for human review before presentation generation.
