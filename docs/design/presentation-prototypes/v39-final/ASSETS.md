# ProposalPilot V39 Final Prototype Assets

## External Assets

No external photos, stock images, third-party icons, screenshots, or reference-image files are embedded. The flower/product image, scan frame, icons, KPI panels, timelines, connectors, and system diagrams are all built from editable PowerPoint-native shapes and text.

## Visual System

- Palette: deep navy, blue, cyan, white, slate surfaces.
- Icons: unified editable shape-and-text icon markers for AI, image, human review, API, DB, CSV, check, and category concepts.
- Image slots: editable placeholder panels labelled as replacement areas, not real customer photos.
- Data integrity: no unsupported actual KPI results, no win probability, no web-production terminology.
