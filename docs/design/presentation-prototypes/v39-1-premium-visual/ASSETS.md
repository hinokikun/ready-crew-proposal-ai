# ProposalPilot V39.1 Premium Visual Assets

## Asset Policy

This prototype uses no external photos, stock images, third-party icons, screenshots, or licensed illustration files. All visible objects are built with editable PowerPoint-native text and shapes.

## Embedded Visual Elements

- Abstract flower/product image built from editable ellipse shapes.
- AI recognition frame built from editable rectangle shapes.
- Floating recognition tags for type, color, grade, condition, candidate result, and human review state.
- Human Review UI mock built from editable cards, pills, icons, and status chips.
- Data flow connectors built from editable PowerPoint shapes.
- KPI dashboard, evaluation states, scope gate, and roadmap built from editable PowerPoint shapes.

## Icons

The deck uses a unified shape-based icon language:

- Image / product photo
- AI engine
- Human reviewer
- API / CSV
- Database / product system
- Learning and operations loop
- Review / approval states

These are not external icon files. They can be edited directly in PowerPoint.

## External References

No external visual asset is embedded. The V39 deck is used only for comparison images generated from local V39 PNGs.
