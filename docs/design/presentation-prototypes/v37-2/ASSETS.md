# ProposalPilot Presentation Prototype Assets

## Scope

Version 37.2 premium presentation visual prototypes for a flower auction AI image-recognition proposal.

## External Assets

No external images, stock photos, icons, fonts, screenshots, or third-party visual files are embedded. All visuals are editable PowerPoint-native text boxes and shapes generated for this prototype.

## Icons and Visual Motifs

- Scan-frame motif: native PowerPoint rectangles.
- Flower/image-recognition motif: native PowerPoint rounded rectangles and text.
- Process arrows and numbered markers: native PowerPoint text boxes and rounded rectangles.
- Architecture and workflow diagrams: editable PowerPoint-native shapes.

## Fonts

The prototype is designed for Noto Sans JP / Inter-like sans-serif presentation use. The generated PPTX uses editable text and may render with the environment's available fallback fonts if Noto Sans JP or Inter is not installed.

## Licensing

Because no external assets are used, there are no third-party image or icon license obligations for this prototype package.

## Reference Handling

The user-provided reference was not copied or traced. It was used only as directional inspiration for strong visual contrast, a meaningful key visual, and clearer diagram emphasis.
