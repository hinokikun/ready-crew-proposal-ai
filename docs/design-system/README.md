# ProposalPilot Design System

ProposalPilot / AI営業秘書のUIを、BtoB SaaSとして一貫した品質で保つためのデザインシステムです。

## 目的

- 一般利用者が迷わず「次へ」で進められること
- 管理者向けの技術情報を通常画面から分離すること
- ProposalPilotブランドの信頼感、速さ、美しさを画面全体に反映すること
- 既存API、DB、権限、生成処理を変えずにUI品質を上げること

## ブランド

- Brand: ProposalPilot
- Product: AI営業秘書
- Tagline: Accelerate Every Proposal.
- Main copy: 営業提案をAIでもっと速く、もっと美しく。

## 構成

- `DESIGN_TOKENS.md`: 色、文字、余白、角丸、影、状態
- `COMPONENTS.md`: 共通UI部品の仕様
- `LAYOUT.md`: アプリ全体のレイアウト
- `ACCESSIBILITY.md`: アクセシビリティ基準
- `RESPONSIVE.md`: 画面幅ごとの表示基準
- `MIGRATION_GUIDE.md`: 既存UIを段階的に移行する手順
