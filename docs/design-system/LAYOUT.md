# Layout

## Desktop

- Left navigation or menu area
- Top context header
- Main content
- Optional right auxiliary panel only when needed

## General User Navigation

一般利用者に優先表示する項目:

- Home
- New Proposal
- Creation History
- AI Review
- Account

管理・診断情報は通常画面へ出さないこと。

## Admin Console

管理者向け機能は「管理コンソール」にまとめること。

- User Management
- Audit Log
- Organization / Workspace
- System Diagnostics
- UAT
- Knowledge
- Learning
- Prompt Studio
- Integrations

## Guided Flow

一般利用者向けの標準操作は7ステップです。

1. 案件入力
2. AI分析
3. 内容確認
4. 提出前チェック
5. 出力
6. AIレビュー・改善
7. 完了
