## 制作時の基本ルール

- 未確認の効果数値を確定コピーに使わない。
- 実顧客情報、APIキー、Password、Token、request_idを素材に含めない。
- 料金、問い合わせ先、会社名、QRコードは未確定なら「要入力」とする。
- AIは提案作成を支援するものであり、人の確認を前提に説明する。
- 競合比較は公開情報で確認できる一般機能に限定する。
