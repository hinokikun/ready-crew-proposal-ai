# ProposalPilot Open Items for Design

ブランド名: ProposalPilot

目的: 制作前に確定が必要な項目を一覧化し、勝手な補完を防ぐ。

| 会社ロゴ | 要入力 | 表紙、裏表紙、Footerに必要 | 要入力 |
| 問い合わせ先メール | 要入力 | CTA、裏表紙、LPに必要 | 要入力 |

| 担当部署 | 要入力 | 問い合わせ導線に必要 | 要入力 |
| 住所 | 要入力 | Footerや会社情報に必要な場合あり | 要入力 |

## 8. 現時点の掲載方針

- 未確定項目は「要入力」として制作担当へ渡す。
- 料金は未確定の場合「お問い合わせ」と記載する。
- 導入実績や効果数値は、根拠が揃うまで掲載しない。
- 会社名、問い合わせ先、QRコードは仮作成しない。
- 実顧客情報は販促物に使用しない。
