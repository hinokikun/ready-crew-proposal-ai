# ProposalPilot Product Roadmap

製品名: AI営業秘書

## Version 1.0

テーマ: 提案書作成と提出前確認の製品化

主な内容:

- 7ステップのかんたん操作フロー
- AI提案書生成
- 要約PPTX、詳細PPTX、見積PDF出力
- Beautiful.ai Prompt API連携
- Quality Gate
- Presentation Review
- Proposal Optimization
- 作成履歴
- ユーザー管理
- admin / manager / member / viewer権限
- Organization / Workspace分離
- 監査ログ
- Maintenance Mode
- Version 1.0製品ドキュメント

到達点:

限定顧客提供、社内利用、受託提案支援に利用可能。

## Version 2.0

テーマ: SaaS販売基盤

候補:

- 料金プラン
- 利用量上限
- 契約管理
- 請求連携
- 外部監視
- SLO / SLA運用
- GitHub Actions Cloud Smoke Test自動化
- PostgreSQL本番移行
- Backup / Restore自動化
- UAT結果のBackend保存
- 管理者向け導入ウィザード
- 顧客別利用状況レポート

到達点:

中小企業向けSaaSとして販売可能な運用基盤を整える。

## Version 3.0

テーマ: 外部業務連携

候補:

- Gmail / Outlook intake
- Slack / Teams intake
- Google Calendar連携
- Google Drive / OneDrive連携
- HubSpot / Salesforce連携
- kintone連携
- OAuth本実装
- External Intake承認フロー強化
- 商談予定から案件化
- 添付資料の安全な要約
- 連携別監査ログ

到達点:

営業担当が手入力しなくても、メール、予定、チャット、資料から案件候補を作れる状態にする。

## Version 5.0

テーマ: Enterprise営業OS

候補:

- SSO / OIDC
- SCIM
- Enterprise監査ログ
- DPA / セキュリティチェックシート
- 高度な権限委任
- 複数法人テナント管理
- AI利用ポリシー管理
- 部門別ナレッジ管理
- 受注結果と提案改善の高度分析
- Executive Dashboard
- API公開
- Partner / Agency向け管理
- 多言語対応
- 専用環境提供

到達点:

大企業、複数部門、複数拠点で運用できる営業AIプラットフォームへ発展させる。
