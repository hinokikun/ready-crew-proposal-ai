# ProposalPilot Exhibition Plan

製品名: AI営業秘書

## 展示会コンセプト

提案書作成に時間がかかる営業チームへ、「案件入力から提案書出力まで5分で見える」体験を提供する。

## ブース構成

### 正面パネル

見出し:

提案書づくりを、AIと一緒に前へ。

サブコピー:

案件情報を貼り付けるだけで、提案書作成、提出前チェック、PPTX/PDF/Beautiful.ai出力まで進められるAI営業秘書です。

表示要素:

- 7ステップの操作フロー
- 出力形式: 要約PPTX / 詳細PPTX / 見積PDF / Beautiful.ai
- 管理者機能: ユーザー管理 / 監査ログ / 診断

### サイドパネル

テーマ:

- 営業担当向け: 次に何を押せばよいか迷わない
- 管理者向け: 権限、履歴、品質を確認できる
- 経営層向け: 提案準備時間を削減し、品質を標準化

### デモ卓

- 1台目: 利用者画面デモ
- 2台目: 管理者画面デモ
- 3台目: Beautiful.ai出力例

## 動画

### 30秒動画

1. 案件情報を貼り付け
2. AIが提案書を作成
3. 提出前チェック
4. PPTX / Beautiful.ai出力
5. 作成履歴に保存

### 90秒動画

1. 営業課題
2. ProposalPilot概要
3. 7ステップ操作
4. Quality Gate
5. Beautiful.ai
6. 管理者診断
7. 導入相談CTA

## 接客トーク

### 呼び込み

提案書作成に時間がかかっていませんか。案件情報を貼り付けるだけで、AIが提案書のたたき台を作るデモを5分でご覧いただけます。

### ヒアリング質問

- 提案書作成にどれくらい時間がかかっていますか
- PowerPoint作成は誰が担当していますか
- 上司レビューや提出前チェックは仕組み化されていますか
- 過去提案の再利用はできていますか
- AI導入で不安な点は何ですか

### クロージング

まずは架空案件または過去の公開可能な案件で、小規模PoCから始められます。営業チーム3から5名で試す導入がおすすめです。
