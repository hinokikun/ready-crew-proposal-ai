# Pilot Dataset Template

実案件評価に使う案件一覧を管理するテンプレート。

## 記入ルール

- 5〜10件を目安に登録する
- 実顧客名は使わず、必要に応じて匿名化する
- 案件概要は要約のみ記載する
- 顧客メール全文、個人情報、秘密情報、APIキー、Password、Tokenは記載しない

## 評価対象の確認

各案件で以下を確認する。

- Legacy Engineの結果
- Strategy v1の結果
- Proposal Quality Report
- Human Review Report
- Comparison Report
- 営業担当のHuman Evaluation
