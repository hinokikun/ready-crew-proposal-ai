# Quality Gate UI

一般利用者向け表示名は「提出前チェック」です。

## 表示方針

- Quality Gateという英語表記は通常モードで前面に出さない
- 完了ボタンは常に見える場所に表示する
- 未チェックがある場合はボタンをdisabledにする
- disabledの理由を「あと○項目の確認が必要です」と表示する
- 完了後はBackendのcompleted / bypassed / download_unlockedをもとに表示する

## チェック項目

- 会社名に誤りがない
- 金額が正しい
- 納期が現実的
- 提案内容に不足がない
- AI推測の項目を確認した
- 社外提出前に上長確認した

## 完了後表示

- 提出前チェックが完了しました
- 完了日時
- 確認者
- ステータス

## 注意

Frontendのローカル状態だけで完了扱いにしません。既存Backend APIの結果をAppShellへ反映します。
