# Version 23.1 Bug Report Template

目的: UAT中に見つかった不具合を、再現しやすい形で記録するためのテンプレートです。

## 記載禁止

以下は絶対に記載しないでください。

- APIキー
- Password
- Token
- 実顧客情報
- 個人情報
- 顧客メール全文
- 顧客の機密案件本文

## 不具合報告

### 発生日時

例: 2026-07-14 10:30

### 確認項目

例: STEP 4 提出前チェック

### 結果

`○ / △ / ×`

### 再現手順

1.
2.
3.

### 期待結果

例: 全チェック後に「提出前チェックを完了する」ボタンが有効になる。

### 実際の結果

例: 全チェック後もボタンが無効のまま。

### Role

例: admin / member / viewer

### Organization

例: Ready Crew

### Workspace

例: 営業部

### Browser

例: Chrome / Edge / Safari

### 画面幅

例: 390px / 768px / 1440px

### Frontend Version

画面に表示される場合のみ記載。

### Backend Version

画面または `/health` で確認できる場合のみ記載。

### request_id

表示される場合のみ記載。表示されない場合は空欄でよい。

### error_type

表示される場合のみ記載。表示されない場合は空欄でよい。

### スクリーンショット名

例: `uat-step4-quality-gate-disabled.png`

### 重大度

- critical
- high
- medium
- low

### 暫定対応

例: 画面を再読み込みして再実行した。
