# Version 23.1 Simple UAT Checklist

目的: Version 23.0 Simple Guided UI の最重要項目だけを短時間で確認するチェックリストです。

結果:

- `○`: 問題なし
- `△`: 利用はできるが改善が必要
- `×`: 利用できない、または重大な問題

## 重大項目

次の項目が `×` の場合は、本番利用不可です。

- ログイン不可
- AI生成不可
- Quality Gate 完了不可
- 出力不可
- 権限漏れ
- Workspace 越境
- Health 異常

## 判定

- 重大項目に `×` が 1 件でもある: 本番利用不可
- 重大項目がすべて `○` で、その他に `△` がある: 条件付きでUAT継続
- すべて `○`: UAT合格候補
