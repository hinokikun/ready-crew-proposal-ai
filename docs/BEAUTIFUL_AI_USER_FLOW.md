# Beautiful.ai User Flow

Beautiful.aiのAPI仕様は変更しません。Version 23.0では一般利用者向け表示だけを簡単にします。

## 利用可能な場合

表示:

Beautiful.aiでプレゼンを作成できます

主ボタン:

Beautiful.aiで提案書を作成

## 利用不可の場合

表示:

Beautiful.aiを利用するには、次の確認が必要です

一般利用者向け条件:

- 提案書を作成する
- 提出前チェックを完了する
- Beautiful.aiを有効にする
- メンテナンス中ではない
- 作成権限がある
- 他の出力処理が終わっている

## 通常モードで表示しない技術条件

- Proposal Ready
- Quality Gate
- Admin
- Role Allowed
- Status API
- Route Found
- Enabled
- Configured
- Mock
- Maintenance
- Other Output Busy

これらは管理者の詳細モードで確認します。
