# Version 37.0 Presentation QA

## 自動確認

PPTX回帰テストは固定スナップショット比較から、品質条件の検査へ変更した。

確認項目:

- PPTXファイルとして開ける
- 外部参照がない
- 壊れたrelationshipがない
- 全スライドにテキストがある
- 全スライドにProposalPilot表記がある
- スライド全体を画像化していない
- 編集可能な図形が十分にある
- 要約版が8から12ページに収まる
- 標準生成の詳細版が20から25ページに収まる
- 画像認識案件でWeb固定文言が混入しない

2. 画像認識案件の詳細PPTX

## 既知の注意

- 生成される本文は案件入力とAI分析結果に依存するため、極端に長い文章が入力された場合はPowerPoint上で一部調整が必要になる可能性がある。
- 詳細版のページ上限整理は、重複タイトルを除外し、主要章を優先する。全文を無制限に保持する用途ではなく、営業提案として読める長さを優先する。
