# 利用者向けマニュアル構成案

## 目的

この文書は、UAT終了後にWord形式の利用者向けマニュアルを作成するための章立てを定義します。

## 想定読者

- 初めてAI営業秘書を使う営業担当者
- 提案書・見積・レビューを確認する上司または管理者
- 社内試験導入後に操作説明を行う研修担当者

## Word化時の編集方針

- 画面確定後にスクリーンショットを追加します。
- UATで文言が変更された場合は、本文ではなく画面の実表示を優先して更新します。
- 機密情報、実顧客名、実案件本文は掲載しません。
- スクリーンショットは架空データで撮影します。

2. 利用者ログイン / 管理者ログインを選ぶ

4. 権限不一致時の案内を確認する
