# ProposalPilot ロゴ候補

対象ブランド: ProposalPilot
製品名: AI営業秘書
Tagline: Accelerate Every Proposal.
ステータス: 候補案。正式採用は人の承認後に決定。

## 制作方針

ProposalPilotのロゴは、営業提案を速く、美しく、安全に進めるBtoB SaaSとしての信頼感を表現します。

共通条件:

- BtoB SaaSらしい信頼感
- 小サイズでも識別可能
- 白黒でも成立
- 特定企業のロゴに似せない
- 脳、ロボット、回路などAIの直接表現を避ける
- 紙飛行機、提案書、Pilot要素を詰め込みすぎない
- 10年以上使える普遍性を優先
- 商標登録可能性は未調査

## Candidate A: Pilot Fold

### コンセプト

`P` の骨格を、折り返しと前進方向のラインで構成した抽象シンボルです。提案書が整理され、次のアクションへ進む感覚を表します。

### 意味

- P: ProposalPilotの頭文字
- Fold: 案件情報から提案書へ整理される流れ
- Forward line: 営業提案が前へ進む状態

### 強み

- アプリアイコンへ展開しやすい
- 小サイズで識別しやすい
- 単色でも成立しやすい
- ProposalPilotらしい固有性を出しやすい

### 注意点

- Pの形状が強いため、既存のP系ロゴとの類似調査が必要
- 角度を強くしすぎると金融・物流系の印象に寄る可能性がある

### 適性

| 項目 | 評価 |
| --- | --- |
| 小サイズ適性 | 高 |
| モノクロ適性 | 高 |
| アプリアイコン適性 | 高 |
| LP適性 | 高 |
| 印刷適性 | 高 |

## Candidate B: Forward Grid

### コンセプト

進行方向を示す幾何学的な2枚のプレートで、提案活動が段階的に前へ進むことを表します。7ステップUIや営業プロセスとの親和性を重視した案です。

### 意味

- 2枚のプレート: AIと人間の協働
- 右上方向: 提案活動の前進
- グリッド感: SaaSらしい構造化と管理性

### 強み

- 「前へ進む」印象が分かりやすい
- アプリ内のステップナビゲーションと相性がよい
- Horizontal / Verticalの展開がしやすい

### 注意点

- 抽象度が高く、ProposalPilot固有の記憶には補助的なwordmarkが必要
- 矢印に寄せすぎると汎用的になりやすい

### 適性

| 項目 | 評価 |
| --- | --- |
| 小サイズ適性 | 中〜高 |
| モノクロ適性 | 高 |
| アプリアイコン適性 | 中 |
| LP適性 | 高 |
| 印刷適性 | 高 |

## Candidate C: Quiet Wordmark

### コンセプト

テキストロゴを中心に、`Pilot` 部分へ控えめなアクセントを加えるミニマル構成です。営業資料、契約書、管理画面、GitHub READMEなどで長期的に使いやすい安定感を重視します。

### 意味

- Proposal: 営業提案の実務感
- Pilot: 迷わず進める案内役
- Accent dot: 次の一手を示す小さな起点

### 強み

- 高級感と信頼感を出しやすい
- 他の販促物に馴染みやすい
- 管理画面や文書で過度に主張しない

### 注意点

- シンボル単体の印象は弱い
- faviconやアプリアイコンには別途簡略シンボルが必要

### 適性

| 項目 | 評価 |
| --- | --- |
| 小サイズ適性 | 中 |
| モノクロ適性 | 高 |
| アプリアイコン適性 | 低〜中 |
| LP適性 | 高 |
| 印刷適性 | 高 |

## 推奨案

現時点の推奨は **Candidate A: Pilot Fold** です。

理由:

- ProposalPilotの頭文字を自然に扱える
- favicon、アプリアイコン、OGP、営業資料へ展開しやすい
- 直接的なAI表現に頼らず、長期利用しやすい
- 小サイズでの識別性が高い

ただし、正式採用前に以下を人が確認してください。

- 類似ロゴ調査
- 商標調査
- 小サイズ表示確認
- 白抜き・単色表示確認
- 実際のアプリ画面での視認性
