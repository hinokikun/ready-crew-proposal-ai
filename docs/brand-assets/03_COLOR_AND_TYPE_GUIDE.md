# ProposalPilot カラー・タイポグラフィガイド

対象ブランド: ProposalPilot
製品名: AI営業秘書

## 1. ブランドカラー

| 名称 | Hex | 用途 |
| --- | --- | --- |
| Proposal Blue | `#155EEF` | 主ボタン、リンク、強調、ロゴアクセント |
| Deep Navy | `#102A43` | 見出し、濃色背景、信頼感の演出 |
| Emerald | `#12B76A` | 成功、完了、チェック済み、前進感 |
| Background | `#F8FAFC` | LP、資料、アプリ背景 |
| Surface | `#FFFFFF` | カード、本文面、余白 |
| Text Primary | `#1D2939` | 本文、見出し |
| Text Secondary | `#667085` | 補足、説明、メタ情報 |

## 2. 補助カラー

| 名称 | Hex | 用途 |
| --- | --- | --- |
| Warning Amber | `#F79009` | 要確認、注意 |
| Error Red | `#D92D20` | エラー、禁止、失敗 |
| Border | `#D0D5DD` | 区切り線、薄い枠 |
| Muted Blue | `#EFF4FF` | Blueの淡色背景 |
| Muted Green | `#ECFDF3` | Greenの淡色背景 |

## 3. カラー比率

推奨比率:

- White / Background: 60%
- Deep Navy / Text: 20%
- Proposal Blue: 12%
- Emerald: 5%
- Warning / Error: 3%以下

Proposal Blueを全面に使いすぎると、SaaSらしい落ち着きが弱くなるため、CTAと重要なアクセントに限定します。

## 4. コントラスト

最低確認:

- 白背景にText Primary
- 白背景にProposal Blue
- Deep Navy背景に白文字
- Proposal Blue背景に白文字
- Emerald背景にDeep Navyまたは白文字

小サイズテキストでは、Proposal BlueよりText Primaryを優先します。

## 5. 推奨フォント

日本語:

- Noto Sans JP
- BIZ UDPGothic
- system-ui fallback

英字:

- Inter
- Segoe UI
- system-ui fallback

PowerPointでフォントがない環境では、Noto Sans JP / Interの代替として、Aptos、Yu Gothic、Segoe UIを使用してもよいです。

## 6. タイポグラフィ

| 用途 | 推奨 |
| --- | --- |
| LP Hero | 48〜72px、太字 |
| スライドタイトル | 34〜48pt、太字 |
| セクション見出し | 24〜32pt、太字 |
| 本文 | 16〜22pt |
| 補足 | 12〜14pt |
| ラベル | 11〜13pt、太字 |

## 7. 表記ルール

- ProposalPilotは英字表記を優先
- AI営業秘書は日本語製品名として併記
- `Accelerate Every Proposal.` はピリオドを含める
- 見出しでは短く、本文では分かりやすく説明する
- 専門用語を出しすぎない

## 8. 印刷時の注意

- Deep Navyの濃色背景では白抜きロゴを使用
- 小さなグレー文字は印刷で薄くなるため、Text Secondaryより濃い色を検討
- QRコードや問い合わせ先が未確定の場合は「要入力」とする
- 会社情報が未確定の場合はダミーであることを明記する
