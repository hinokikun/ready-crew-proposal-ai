# ProposalPilot OGP / SNS画像仕様

対象: ProposalPilot / AI営業秘書
ステータス: 公開前テンプレート

## 1. 基本方針

OGP / SNS画像は、ProposalPilotの信頼感、未来感、営業支援プロダクトとしての実用性を伝えるための公開用画像です。

必須要素:

- ProposalPilot
- AI営業秘書
- Accelerate Every Proposal.
- 営業提案をAIでもっと速く、もっと美しく。
- UIモックまたは抽象的なプロダクト画面
- ブランドカラー

会社名、製品URL、問い合わせ先が未確定の場合は「要入力」または非表示とします。

## 2. サイズ

| 用途 | サイズ | 比率 |
| --- | --- | --- |
| OGP | 1200x630 | 1.91:1 |
| X / SNS | 1200x675 | 16:9 |
| LinkedIn | 1200x627 | 1.91:1 |
| YouTube Thumbnail | 1280x720 | 16:9 |

## 3. レイアウト

推奨構成:

1. 左側にブランド名、製品名、メインコピー
2. 右側にUIモックまたはカード型の抽象画面
3. 背景はBackgroundまたはDeep Navy
4. CTAあり版では下部に短いCTAを配置
5. CTAなし版では余白を広く保つ

## 4. CTAあり版

CTA例:

- 無料デモを見る
- 製品資料ダウンロード
- 要入力

CTAが未確定の場合は、公開版では非表示にします。

## 5. CTAなし版

汎用利用向けに、CTAを含まない版を用意します。

用途:

- SNS共有
- GitHub README
- プレス素材
- 営業資料の表紙
- 展示会告知

## 6. 禁止事項

- 実顧客名を載せない
- 実メールアドレスを載せない
- 実電話番号を載せない
- APIキー、Token、管理画面の内部情報を写さない
- 競合サービスのロゴを並べない
- Beautiful.ai、PowerPoint等の他社ロゴをブランド資産として混在させない
- 商標登録済みと誤認される表記をしない

## 7. 書き出し

推奨:

- PNG: 公開用
- PDF: 確認用
- PPTX: 編集用
- SVG: ロゴ部分のみ

ファイル名例:

- `ProposalPilot_OGP_1200x630`
- `ProposalPilot_SNS_1200x675`
- `ProposalPilot_LinkedIn_1200x627`
- `ProposalPilot_YouTube_1280x720`

## 8. 確認項目

- [ ] 1200x630で文字切れなし
- [ ] モバイル表示で主コピーが読める
- [ ] 日本語文字化けなし
- [ ] ブランドカラーに準拠
- [ ] CTAあり / なしが分かれている
- [ ] 未確定項目は要入力または非表示
- [ ] 実顧客情報なし
- [ ] 秘密情報なし
