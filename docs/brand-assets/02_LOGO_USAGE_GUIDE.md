# ProposalPilot ロゴ利用ガイド

対象ブランド: ProposalPilot
ステータス: 候補ロゴ向け利用ルール。正式採用後に確定版へ更新。

## 1. 基本表記

正式表記:

- ProposalPilot
- AI営業秘書
- Accelerate Every Proposal.

表記ルール:

- `ProposalPilot` の大文字小文字を変更しない
- `Proposal Pilot` のようにスペースを入れない
- `PROPOSALPILOT` のような全大文字表記は見出し用途以外では避ける
- 日本語製品名は `AI営業秘書` と表記する
- Taglineは英語表記 `Accelerate Every Proposal.` を使用する

## 2. ロゴ構成

| 種別 | 用途 |
| --- | --- |
| 横組みロゴ | LP、パンフレット、製品カタログ、営業資料、GitHub README |
| 縦組みロゴ | 表紙、展示会パネル、広い余白がある資料 |
| シンボル単体 | favicon、アプリアイコン、SNSアイコン、スライドの小さな署名 |
| 白抜き版 | Deep Navy背景、写真上、濃色背景 |
| モノクロ版 | 契約書、白黒印刷、FAX、内部資料 |

## 3. クリアスペース

ロゴ周囲には、シンボル高さの25%以上の余白を確保してください。

推奨:

- 横組みロゴ: ロゴ高さの0.25倍以上
- シンボル単体: シンボル幅の0.2倍以上
- Tagline併記時: ロゴとTaglineの間に8px以上

## 4. 最小サイズ

| 用途 | 最小サイズ |
| --- | --- |
| 横組みロゴ Web | 幅140px以上 |
| 横組みロゴ 印刷 | 幅35mm以上 |
| シンボル Web | 24px以上 |
| シンボル favicon | 16px以上 |
| 縦組みロゴ | 幅80px以上 |

16px表示では細部が潰れるため、faviconはシンボル単体の簡略版を使用します。

## 5. 背景ごとの利用

| 背景 | 推奨 |
| --- | --- |
| 白背景 | カラー版 |
| Background `#F8FAFC` | カラー版 |
| Deep Navy `#102A43` | 白抜き版 |
| Proposal Blue `#155EEF` | 白抜き版 |
| 写真背景 | 原則として白またはDeep Navyのプレート上に配置 |
| 複雑な背景 | 使用しない |

## 6. 禁止事項

- 色を任意に変更しない
- 縦横比を変えない
- 回転しない
- 影や縁取りを追加しない
- グラデーションを過度に追加しない
- ロゴ内の文字間を変更しない
- シンボルとwordmarkの距離を変更しない
- 他社ロゴと並べる際に一体化して見せない
- Beautiful.ai、PowerPoint等の他社ロゴをProposalPilotブランド資産へ混在させない
- 商標登録済みであるかのように表記しない

## 7. AI営業秘書との併記

基本形:

```text
ProposalPilot
AI営業秘書
```

横組み資料では `ProposalPilot | AI営業秘書` のように区切ってもよいですが、アプリ内では過度に長くならないよう、画面上部では `ProposalPilot` を優先します。

## 8. Taglineとの併記

Taglineは以下のように、ロゴの補助情報として扱います。

```text
ProposalPilot
Accelerate Every Proposal.
```

注意:

- Taglineをロゴの一部として固定しない
- 小サイズ表示ではTaglineを省略する
- 日本語コピーとTaglineを同時に表示する場合は、Taglineを小さく扱う

## 9. 正式採用前の注意

本ガイドは候補ロゴ向けです。正式利用前に、商標・類似ロゴ調査、法務確認、営業資料での視認確認を実施してください。
