# Version52 Known Limitations

- Proposal Preview is admin-only.
- Proposal Preview is not saved to DB.
- Proposal Preview does not create PPTX.
- Proposal Preview does not call Beautiful.ai.
- Proposal Preview does not send email.
- Proposal Preview does not write creation history.
- Proposal Preview does not feed learning or analytics dashboards.
- Performance measurements in this release are local Mock AI measurements, not cloud OpenAI measurements.
- Monitoring is documented but not implemented.
- Human Review is displayed, but approval workflow is not connected to output generation.
