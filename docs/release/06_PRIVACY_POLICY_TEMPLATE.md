制定日: 要入力

## 1. 基本方針

要入力は、ProposalPilot / AI営業秘書の提供にあたり、利用者の情報を適切に取り扱います。

## 2. 取得する情報

本サービスでは、以下の情報を取得する場合があります。

| 情報 | 例 | 取得有無 |
| --- | --- | --- |
| アカウント情報 | 氏名、所属、Role、Organization、Workspace | 要入力 |
| 連絡先情報 | 業務用連絡先 | 要入力 |
| 利用履歴 | ログイン、案件作成、出力、Quality Gate、Beautiful.ai生成 | 要入力 |
| 技術情報 | ブラウザ、IPの一部、request_id、エラー種別 | 要入力 |
| 入力情報 | 案件メモ、提案作成に必要な要約情報 | 要入力 |
| 決済情報 | 利用プラン、請求情報 | 要入力 |

## 3. 利用目的

取得した情報は、以下の目的で利用します。

- 本サービスの提供
- アカウント管理
- Organization / Workspaceごとのアクセス制御
- 提案書、PPTX、PDF、Beautiful.ai出力の提供
- 障害調査とサポート
- セキュリティ確保
- 利用状況の集計
- サービス改善
- 契約、請求、連絡

## 4. 保存しない情報

本サービスでは、以下を平文で保存しない方針とします。

- Password
- APIキー
- Token
- Authorizationヘッダー
- 外部サービスの秘密情報

また、Analytics、Knowledge、監査ログには、顧客本文全文や不要な機密情報を保存しない運用とします。

本サービスは、提供や運用のために外部サービスを利用する場合があります。

| 外部サービス | 用途 | 利用有無 |
| --- | --- | --- |
| OpenAI | AI生成 | 要入力 |
| Beautiful.ai | プレゼンテーション生成 | 要入力 |
| Vercel | Frontend配信 | 要入力 |
| Render | Backend稼働 | 要入力 |

| Google Search Console | 検索状態確認 | 要入力 |

外部サービスへ送信する情報は、機能提供に必要な範囲に限定します。

## 6. 第三者提供

要入力は、以下の場合を除き、本人の同意なく個人情報を第三者へ提供しません。

- 法令に基づく場合
- 本人の同意がある場合
- 業務委託先へ必要な範囲で提供する場合
- 人の生命、身体または財産の保護のため必要な場合

## 7. 業務委託

本サービスの提供に必要な範囲で、情報の取り扱いを外部事業者へ委託する場合があります。委託先には適切な管理を求めます。

## 8. 安全管理措置

本サービスでは、以下の安全管理を行います。

- Roleに基づくアクセス制御
- Organization / Workspace分離
- Passwordのハッシュ保存
- 秘密情報の非表示
- 監査ログ
- 重要操作の記録
- 不要な情報保存の抑制
