# Pilot User Guide

AI営業秘書を初めて使う方向けの社内試験利用ガイドです。

## 使い方

1. ログインする
2. 初回チェックリストを確認して開始する
3. Ready Crewの案件メール、議事録、ヒアリングメモを貼る
4. AI Workspaceの進行を確認する
5. 提案書作成結果を確認する
6. 提出前品質ゲートを確認する
7. 要約PPTをダウンロードする
8. 社外提出前に必ず人が最終確認する

## 入力時の注意

- 顧客の機密情報を必要以上に入力しない
- パスワード、APIキー、認証コードは入力しない
- 個人情報は必要最小限にする
- AIの推測ラベルが付いた内容は必ず確認する

## 完成後に確認すること

- 会社名や担当者名に誤りがないか
- 見積金額が妥当か
- 納期が現実的か
- 実績や事例表記に誤りがないか
- 上司レビューや品質ゲートを確認したか

## 問題が起きたら

画面に表示される日本語メッセージに従ってください。

- Backend未接続: 少し待って再読み込みし、改善しなければ管理者へ連絡
- OpenAI API上限: 時間を置いて再実行、または管理者へ連絡
- PPT/PDF失敗: 再試行し、改善しなければ管理者へ連絡
- 権限不足: Pilot対象になっているか管理者へ確認

## フィードバック

作成完了後に「この提案書は使えそうですか？」へ回答してください。

コメント欄には顧客本文や案件本文を貼り付けないでください。使いやすさ、分かりにくかった場所、改善してほしい点だけを書いてください。
